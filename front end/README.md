# Professional Portfolio Website

A responsive, recruiter-friendly portfolio website designed to highlight personal brand, skills, and projects. Built with clean HTML, CSS, and JavaScript.

## Features

- **Responsive Design:** Fully optimized for mobile, tablet, and desktop devices.
- **Clean Navigation:** Sticky navbar with smooth scrolling to sections.
- **Project Showcase:** Dedicated section for projects with GitHub and live demo links.
- **Professional Layout:** Sections for About, Skills, Experience, Education, and Contact.

## Tech Stack

- **HTML5:** Semantic structure.
- **CSS3:** Flexbox, Grid, and CSS Variables for styling.
- **JavaScript:** Mobile menu interactivity.

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   ```
2. **Open the project:**
   Navigate to the project folder and open `index.html` in your browser.

## Customization

1. Open `index.html` and replace placeholders (e.g., `[Your Name]`, `[Role]`) with your real information.
2. Add your project screenshots to an `images` folder and update the `src` attributes in the Projects section.
3. Update the links in the Contact section to point to your actual social profiles.

## Deployment

This project is ready to be deployed on platforms like:
- **GitHub Pages:** Go to Settings > Pages > Select Branch (main) > Save.
- **Netlify / Vercel:** Drag and drop the project folder or connect your GitHub repository.
